# nodejs-cli
A nodejs command line tool for daily usage

# Usage
<code>
  docker run --rm -it oopschen/nodejs-cli
</code>

## NPM
<code>
  docker run --rm -it --entrypoint=npm oopschen/nodejs-cli install
</code>
